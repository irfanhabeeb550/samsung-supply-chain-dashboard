# Samsung Supply Chain & Logistics Dashboard

An end-to-end Power BI dashboard tracking Samsung's supply chain performance, from supplier lead times through inventory, shipment, and customer revenue.

## Key Metrics
- Gross Revenue, Net Revenue, Profit, Profit Margin
- Perfect Order Rate
- Average Lead Time by Supplier
- Inventory Quantity by Product
- Shipment Delay by Carrier
- Revenue by Customer Platform (Amazon, Flipkart, Best Buy, Samsung.com, MediaMarkt)

## Tools Used
Power BI Desktop, DAX, Power Query

## Data Model
Star schema with one fact table (Orders) and five dimension tables (Suppliers, Products, Inventory, Carriers, Customers), connected through a proper date table for time intelligence.
